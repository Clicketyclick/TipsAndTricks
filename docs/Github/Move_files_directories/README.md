
## Move and rename files and directories on Github

### Rename a file

Simply edit the file and change the file name.

### Moving files to new locations

To move a file up, place your cursor at the beginning of the filename field and either:

- Type `../` to jump up a whole directory level.
- Press backspace to move up a directory, while keeping the parent folder’s name around for quick changes.

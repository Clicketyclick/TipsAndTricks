- <kbd>⊞ Windows</kbd> U+229E ⊞ SQUARED PLUS
- <kbd>⌘ Command</kbd>
- <kbd>⇧ Shift</kbd>
- <kbd>⌥ Option</kbd>
- <kbd>⇪ Caps Lock</kbd>
- <kbd>Tab ↹</kbd> Tablular left and right

- <kbd>&darr;</kbd> &amp;darr; Down arrow
- <kbd>&larr;</kbd> &amp;larr; Left arrow
- <kbd>&rarr;</kbd> &amp;rarr; Right arrow
